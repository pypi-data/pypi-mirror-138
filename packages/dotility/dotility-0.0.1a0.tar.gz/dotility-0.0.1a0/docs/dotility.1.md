# NAME
dotility -- Automatically set up personal dotfiles in a blink

# SYNOPSIS
**dotility**

# DESCRIPTION
**dotility** makes setting up dotfiles on your system easy.

# OPTIONS  
**-h**, **--help**
: Print help text.

**-V**, **--version**
: Print version.

**-C**, **--config[=]<config>**
: Use \<config\> as a config file.
